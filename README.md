# Nonlinear Optimization

A simple web application for **nonlinear optimization** and visualization


## how to run project

- Install requirements
	> pip install -r requirements.txt

- Run web application
	> app.py
	
- Open this link
	>http://127.0.0.1:5000


## Algorithms

- **Gradient**
- **Newton**
- **BFGS**
- **Genetic**
- **Monte Carlo**
